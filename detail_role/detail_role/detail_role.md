```

│                                                                          │

│   ───────────────────────────────────────────────────────────────────   │
│                                                                          │
│   🎯 MISSION GLOBALE                                                     │
│   ─────────────────                                                     │

│   de chaque citoyen ivoirien dans l'écosystème administratif.            │
│   Il est l'interface unique entre l'humain et la machine                 │

│                                                                          │
│   📋 MISSIONS DÉTAILLÉES                                                 │
│   ──────────────────────                                                 │
│                                                                          │

│   ┌─────────────────────────────────────────────────────────────────┐   │
│   │                                                                  │   │
│   │  • Stocker de manière sécurisée les données d'identité :         │   │

│   │    - Date et lieu de naissance                                   │   │

│   │    - Numéro de téléphone vérifié                                 │   │
│   │    - Adresse de résidence                                        │   │
│   │    - Photo d'identité                                            │   │
│   │                                                                  │   │

│   │    l'inscription (croisement des données)                        │   │
│   │                                                                  │   │
│   │  • Gérer le cycle de vie de l'identité numérique :               │   │
│   │    - Création (inscription)                                      │   │
│   │    - Mise à jour (changement d'adresse, de nom)                  │   │
│   │    - Suspension (en cas de décès)                                │   │
│   │    - Suppression (droit à l'oubli)                               │   │
│   │                                                                  │   │
│   │  • Protéger les données contre tout accès non autorisé           │   │
│   │    (chiffrement, authentification forte)                         │   │
│   │                                                                  │   │
│   └─────────────────────────────────────────────────────────────────┘   │
│                                                                          │
│   MISSION 2 : CONNAISSANCE DES DROITS DU CITOYEN                        │
│   ┌─────────────────────────────────────────────────────────────────┐   │
│   │                                                                  │   │
│   │  • Maintenir un registre actualisé de tous les droits            │   │
│   │    ouverts au citoyen :                                          │   │
│   │    - Prestations familiales (CNPS)                               │   │

│   │    - Droits à la retraite                                        │   │
│   │                                                                  │   │
│   │  • Vérifier périodiquement l'éligibilité à de nouveaux           │   │
│   │    droits (ex: naissance → allocations)                          │   │
│   │                                                                  │   │
│   │  • Alerter le citoyen lorsqu'un droit est sur le point           │   │
│   │    d'expirer (ex: renouvellement CNI, CMU)                       │   │
│   │                                                                  │   │
│   │  • Identifier les droits non réclamés et les suggérer            │   │
│   │    au citoyen ("Saviez-vous que vous avez droit à...?")          │   │
│   │                                                                  │   │
│   └─────────────────────────────────────────────────────────────────┘   │
│                                                                          │
│   MISSION 3 : GESTION DE L'HISTORIQUE ADMINISTRATIF                      │
│   ┌─────────────────────────────────────────────────────────────────┐   │
│   │                                                                  │   │
│   │  • Conserver l'historique complet et chronologique de toutes     │   │
│   │    les démarches effectuées                                      │   │
│   │                                                                  │   │
│   │  • Archiver tous les documents obtenus :                         │   │
│   │    - Extraits d'acte de naissance                                │   │
│   │    - Certificats de mariage                                      │   │
│   │    - Attestations fiscales                                       │   │
│   │    - Diplômes                                                    │   │
│   │    - etc.                                                        │   │
│   │                                                                  │   │
│   │  • Permettre la recherche rapide dans l'historique               │   │
│   │    (par date, type, institution)                                 │   │
│   │                                                                  │   │
│   │  • Permettre le téléchargement des documents archivés            │   │
│   │    à tout moment                                                 │   │
│   │                                                                  │   │
│   └─────────────────────────────────────────────────────────────────┘   │
│                                                                          │
│   MISSION 4 : INTERFACE UTILISATEUR INTELLIGENTE                         │
│   ┌─────────────────────────────────────────────────────────────────┐   │
│   │                                                                  │   │
│   │  • Présenter les événements de vie de manière simple             │   │
│   │    et intuitive (icônes, texte clair)                            │   │
│   │                                                                  │   │
│   │  • Guider l'utilisateur étape par étape dans la saisie           │   │
│   │    des informations                                              │   │
│   │                                                                  │   │
│   │  • Traduire le langage administratif complexe en                 │   │
│   │    français simple et compréhensible                             │   │
│   │                                                                  │   │
│   │  • Adapter l'interface au profil de l'utilisateur :              │   │
│   │    - Mode simplifié pour utilisateurs peu alphabétisés           │   │
│   │    - Mode vocal (futur)                                          │   │
│   │    - Langues locales (futur)                                     │   │
│   │                                                                  │   │
│   │  • Valider les informations saisies avant envoi                  │   │
│   │    (format, cohérence, champs obligatoires)                      │   │
│   │                                                                  │   │
│   └─────────────────────────────────────────────────────────────────┘   │
│                                                                          │
│   MISSION 5 : RÉCEPTION ET GESTION DES NOTIFICATIONS                     │
│   ┌─────────────────────────────────────────────────────────────────┐   │
│   │                                                                  │   │
│   │  • Recevoir les notifications de l'Agent Notificateur           │   │
│   │                                                                  │   │
│   │  • Afficher les notifications dans l'application                 │   │
│   │    avec le bon niveau de priorité                                │   │
│   │                                                                  │   │
│   │  • Permettre à l'utilisateur de :                                │   │
│   │    - Marquer comme lu                                            │   │
│   │    - Archiver                                                    │   │
│   │    - Agir directement depuis la notification                     │   │
│   │      (ex: "Document demandé" → ouvrir l'upload)                  │   │
│   │                                                                  │   │
│   │  • Respecter les préférences de notification du citoyen          │   │
│   │    (heures de silence, canaux préférés)                          │   │

│   • Taux d'abandon de formulaire < 10%                                  │

│   • Délai de réception des notifications < 30 secondes                  │
│                                                                          │
│   📦 VERSION : MVP                                                       │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

│   │                                                                  │   │
│   └─────────────────────────────────────────────────────────────────┘   │
│                                                                          │
│   MISSION 2 : VÉRIFICATION AUTOMATISÉE DES DOSSIERS                      │
│   ┌─────────────────────────────────────────────────────────────────┐   │
│   │                                                                  │   │
│   │  • Vérifier la complétude du dossier :                           │   │
│   │    - Tous les champs obligatoires sont remplis                   │   │
│   │    - Tous les documents requis sont fournis                      │   │
│   │    - Les documents sont lisibles (qualité minimale)              │   │
│   │                                                                  │   │
│   │  • Vérifier la cohérence des informations :                      │   │
│   │    - Les dates sont cohérentes entre elles                       │   │
│   │    - Les noms correspondent entre les documents                  │   │
│   │    - Pas de contradiction entre les pièces                       │   │
│   │                                                                  │   │
│   │  • Croiser avec d'autres sources :                               │   │
│   │    - Vérifier l'identité auprès de l'Agent ONECI                 │   │
│   │    - Vérifier la situation fiscale auprès de l'Agent Fisc        │   │
│   │    - Vérifier les droits sociaux auprès de l'Agent CNPS          │   │
│   │                                                                  │   │

│   │    - Fausses déclarations                                        │   │
│   │    - Documents falsifiés (analyse d'image basique)               │   │
│   │    - Incohérences flagrantes                                     │   │
│   │                                                                  │   │
│   └─────────────────────────────────────────────────────────────────┘   │
│                                                                          │

│   │    - Présenter une synthèse du dossier                           │   │
│   │    - Mettre en évidence les points d'attention                   │   │
│   │    - Suggérer les options possibles avec leurs bases légales     │   │

│   │                                                                  │   │

│   │    - Quand                                                    │   │

│   │    - Avec possibilité de recours                                 │   │
│   │                                                                  │   │

│                                                                          │
