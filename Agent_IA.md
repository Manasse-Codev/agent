
### Catégorie 1 : Les Agents Personnels (Centrés sur l'Usager)

Ce sont les représentants numériques des individus ou entités dans le système.

**1. Agent Citoyen**
- **Rôle :** Représenter et assister un citoyen dans toutes ses interactions avec l'administration.
- **Détail :** Il est le point d'entrée unique. Il connaît l'identité, la situation familiale, les droits et l'historique des démarches de son citoyen. Il reçoit les notifications, initie les demandes et traduit le langage administratif en actions simples.

**2. Agent Entrepreneur**
- **Rôle :** Représenter un créateur d'entreprise ou une société.
- **Détail :** Il gère le cycle de vie complet d'une entreprise : création, modification de statuts, déclarations fiscales, demandes d'agrément. Il dialogue avec les agents des impôts, de la CNPS et du CEPICI.

**3. Agent Agent Public (ou Fonctionnaire)**
- **Rôle :** Assister l'agent de l'État humain dans son travail d'instruction et de décision.
- **Détail :** Il ne remplace pas forcément l'humain. Il prépare les dossiers, vérifie la cohérence, propose une décision et laisse la validation finale à l'humain pour les cas complexes. Il est le "bras droit" numérique du fonctionnaire.

### Catégorie 2 : Les Agents Institutionnels (Services Publics)

Chaque administration clé est représentée par un agent capable de traiter des demandes de manière autonome.

**4. Agent État Civil (Mairie)**
- **Rôle :** Gérer et délivrer tous les actes d'état civil de manière sécurisée.
- **Détail :** Interrogé pour un extrait de naissance, un certificat de décès ou un acte de mariage. Il ne délivre pas une copie à l'usager, mais transmet un justificatif numérique certifié directement à l'agent demandeur.

**5. Agent Identité (ONECI)**
- **Rôle :** Gérer les pièces d'identité nationales (CNI, Passeport).
- **Détail :** Il pilote le processus de demande, de renouvellement et de suivi de fabrication. Il peut authentifier un citoyen dans l'écosystème sur la base de ses données biométriques.

**6. Agent Justice**
- **Rôle :** Gérer l'accès aux informations judiciaires.
- **Détail :** Il est le seul habilité à fournir un extrait de casier judiciaire (bulletin n°3) de manière sécurisée à un autre agent autorisé.

**7. Agent Fisc (Direction Générale des Impôts)**
- **Rôle :** Gérer l'identifiant fiscal unique et les déclarations.
- **Détail :** Il attribue l'identifiant fiscal, reçoit les déclarations, calcule l'impôt et peut fournir une attestation de situation fiscale à un autre agent habilité.

**8. Agent Social (CNPS / CMU)**
- **Rôle :** Gérer les droits sociaux, les prestations familiales et la couverture maladie.
- **Détail :** Il traite les immatriculations d'employeurs/employés, les déclarations de salaires, et les demandes de prestations (allocations familiales, pension, CMU). Il est proactif pour signaler des droits ouverts.

**9. Agent Domaine et Foncier**
- **Rôle :** Gérer les informations cadastrales et les titres de propriété.
- **Détail :** Interrogé pour une demande de permis de construire ou une mutation, il fournit l'extrait cadastral et vérifie la situation juridique d'un terrain.

**10. Agent Éducation (MEN)**
- **Rôle :** Gérer les inscriptions et les diplômes.
- **Détail :** Il traite les inscriptions scolaires en ligne et peut délivrer une attestation de diplôme vérifiée pour un dossier de concours ou d'embauche.

### Catégorie 3 : Les Agents de Coordination et d'Infrastructure

Ce sont les chefs d'orchestre et les agents techniques qui assurent le fonctionnement de l'écosystème.

**11. Agent Coordonnateur (ou Orchestrateur)**
- **Rôle :** Traduire un "événement de vie" en une série de démarches coordonnées.
- **Détail :** C'est l'agent le plus important. Lorsqu'un Agent Citoyen signale "je déménage", "j'ai un enfant", "je crée une entreprise", l'Agent Coordonnateur prend le relais. Il décompose l'événement en procédures, identifie tous les Agents Services concernés, collecte les justificatifs une seule fois et suit l'avancement global. C'est lui qui met en œuvre le principe "Dites-le nous une fois".

**12. Agent Notaire / Authentificateur**
- **Rôle :** Horodater, sceller et rendre infalsifiable chaque transaction et document échangé.
- **Détail :** Il ne traite pas de démarche. C'est un agent technique de confiance. Chaque fois qu'un document est émis (extrait de naissance par l'Agent Mairie), l'Agent Notaire y appose une signature électronique et une preuve d'intégrité pour garantir son authenticité.

**13. Agent Paiement (Trésor Public / Mobile Money)**
- **Rôle :** Gérer de manière sécurisée tous les flux financiers administratifs.
- **Détail :** Quand un Agent Citoyen doit payer un timbre fiscal ou des frais de dossier, c'est cet agent qui génère l'ordre de paiement, le relie à la démarche, et notifie l'Agent Service du succès du paiement, le tout sans que l'usager ait à joindre un reçu de paiement.

**14. Agent Surveillant (Chien de garde)**
- **Rôle :** Surveiller le respect des délais légaux et des règles de l'écosystème.
- **Détail :** Il est le garant de la performance et de la lutte contre la corruption. Il mesure le temps de traitement de chaque Agent Service. Si un agent dépasse le délai légal, il envoie une alerte au responsable. Il rend compte de la "santé" de l'écosystème via des tableaux de bord publics.

**15. Agent Notificateur (Messager)**
- **Rôle :** Gérer le système de communication et de notifications multicanal.
- **Détail :** Il est le facteur de l'écosystème. Il reçoit les messages des autres agents et les achemine vers l'Agent Citoyen via le canal approprié (SMS, notification mobile, email). Il s'assure que l'usager est informé en temps réel de l'avancement de ses dossiers.
